# Baiet Metal Website Deployment Guide

Your high-performance Next.js 15 website is ready!

## 🚀 One-Click Deployment (Recommended)
1. **GitHub**: Create a new repository on GitHub (e.g., `baiet-metal-web`).
2. **Push Code**:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin YOUR_GITHUB_URL
   git push -u origin main
   ```
3. **Vercel**: Go to [Vercel](https://vercel.com), click **Add New Project**, and import your GitHub repo. It will automatically detect Next.js and deploy.
